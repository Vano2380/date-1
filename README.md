# 💖 Мини-приложение "Особое приглашение"

Романтичное веб-приложение для приглашения девушки на свидание в необычной форме.

## 📁 Файлы проекта

- `index.html` - главная страница приложения
- `style.css` - стили и анимации
- `script.js` - логика работы приложения

## 🚀 Как использовать

1. Откройте файл `index.html` в браузере
2. Отправьте ссылку на приложение девушке
3. Она заполнит форму с предпочтениями:
   - Тип прогулки (парк, город, кафе, набережная, закат)
   - Удобное время (свой вариант)
   - Место встречи
4. После нажатия кнопки "Да, пойдем!" или "Может быть..." она увидит свой ответ


## 📧 Как получить ответ

Ответ придет на email  с темой "💖 Новый ответ на приглашение!"

В письме будет:
- Ответ (Да / Может быть)
- Тип прогулки
- Время (её вариант)
- Место встречи
- Дата и время ответа

### ⚠️ Важно для первого запуска!

При первом использовании сервис FormSubmit отправит * на твой email. Нужно:
1. Открыть письмо от FormSubmit
2. Кликнуть на кнопку подтверждения
3. После этого все ответы будут приходить автоматически!

## ✨ Особенности

- Красивый градиентный дизайн с анимациями
- Плавающие сердечки на фоне
- Адаптивная верстка для мобильных устройств
- Автоматическая отправка ответов на email
- Два варианта ответа: "Да" и "Может быть"
- Девушка может сама указать удобное время
