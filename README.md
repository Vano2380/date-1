# 💖 Приложение "Приглашение на прогулку"

Веб-приложение для приглашения на прогулку с автоматическим сохранением ответов в Google Sheets.

## 📁 Структура проекта

**Для приглашения:**
- `index.html` - форма приглашения (отправляй эту ссылку)
- `style.css` - стили формы
- `script.js` - логика формы и отправка в Google Sheets

**Панель администратора:**
- `admin.html` - панель просмотра всех ответов
- `admin-style.css` - стили админки
- `admin-script.js` - логика админки и загрузка из Google Sheets

## 🚀 Как работает

### Для девушки (без регистрации):
1. Открывает ссылку: `https://vano2380.github.io/date-1/`
2. Выбирает:
   - Тип прогулки
   - Дату и время (через календарь)
   - Место встречи
3. Нажимает кнопку:
   - **"Да"** → видит "✅ Правильный вариант!"
   - **"Нет"** → всплывает окно "❌ Неверный вариант!"
4. Ответ автоматически сохраняется в Google Sheets

### Для тебя (просмотр ответов):
1. **Вариант 1:** Открой Google таблицу "Ответы на приглашение"
   - Работает на любом устройстве (компьютер, телефон)
   - Ответы отображаются в реальном времени

2. **Вариант 2:** Открой панель администратора
   - Ссылка: `https://vano2380.github.io/date-1/admin.html`
   - Красивое отображение всех ответов
   - Автообновление каждые 5 секунд
   - Статистика ответов

## 📊 Google Sheets интеграция

Все ответы автоматически сохраняются в Google таблицу со следующими данными:
- ⏰ Время ответа
- ✅ Ответ (Да/Нет)
- 🌳 Тип прогулки
- 📅 Дата и время прогулки
- 📍 Место встречи
